My Languages

There are many, many great languages to choose from. Below you'll find the languages that I have decided to learn thus far in my Software Development journey!

HTML5

HTML5 is the lastest update of plain HTML and provides new tag semantics that help organize the structure of a HTML page. HTML5 was used in the creation of this website!

CSS3

CSS3 is the latest version of plain CSS that that provides the styling of webpages, such as color, sizing, fonts, etc. CSS3 and CSS Grid was used in the creation of this website!

JavaScript

JavaScript is a scripting language that enables the HTML to do certain tasks that would be much more difficult without the help of JavaScript. JavaScript was not used in the creation of this website!

C++

C++ is a high-level language that implements the Object-Oriented Programming features and is popularly used for game development. C++ was not used used in the development process of this website!
